How AI Works

AI does not literally "know" what it is seeing or saying. It turns information into numbers, scores possible answers, and chooses likely outputs. That is powerful, but it also explains why AI can hallucinate.

Part 1: Image classification as evidence

The image model acts like a group of tiny detectives. Each detective looks for one kind of clue. The clues become votes, and the votes become probabilities.

How image scores become probabilities

The detector votes create scores for each label. A softmax calculation turns those scores into a 0 to 100% scale, so CAT, DOG, FOX, and OTHER can be compared.

P(CAT) = eCAT score eCAT + eDOG + eFOX + eOTHER

What just happened?

Tiny detectors looked for patterns such as ears, eyes, texture, and shape.

Each clue cast a vote

Strong matches pushed an answer up. Bad matches pushed it down.

Votes became probabilities

The model compared all possible labels and picked the highest probability.

Confidence is still a score

A high number means the model is confident, not guaranteed to be correct.

Part 2: Language models predict the next token

ChatGPT-style models do something similar, but with words or word-pieces instead of image labels. A token is a small chunk of text: it might be a whole word, part of a word, a number, or a punctuation mark. The model predicts which token is most likely to come next. The temperature setting below changes this probability distribution, so watch the bars shift as you choose a preset.

Prompt

Key idea: Unless it is connected to tools or sources, the model does not first check a truth database. It produces likely continuations based on patterns learned during training.

Probability distribution

There is not one possible next word. There are many, each with a probability.

Temperature changes these bars

Lower temperature makes the top answer dominate. Higher temperature gives less likely answers more room to appear.

Token temperature math

Temperature reweights every possible next token, then the model normalizes the numbers back into probabilities.

p T = p 1 / T Σ p 1 / T

Why text predictions can hallucinate

The model is still choosing likely tokens, not automatically checking a truth database. Temperature changes how adventurous that choice is: lower temperature is more predictable, and higher temperature is more creative but riskier.
